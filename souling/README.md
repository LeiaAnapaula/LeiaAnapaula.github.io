# Souling - Operating System for the Human Soul

A transformative platform connecting hypnotherapists with patients through AI-enhanced therapeutic experiences, featuring inner child avatars and personalized hypnotherapy sessions.

## Features
- Hypnotherapist & Patient matching platform
- AI-enhanced hypnotherapy audio generation
- Inner child avatar visualization
- Session management & tracking
- Memory reconstruction & therapeutic journaling
- Real-time audio enhancement

## Tech Stack
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- AI: Anthropic Claude API
- Audio: Web Audio API
- 3D Avatars: Three.js
